# CharacterSheet

This project is a Webapp centered on Scala Play Stack technologies used to handle an RPG Character Sheet.

### Used technologies

| Techno            | Version |
|:------------------|:--------|
| Scala             | 2.12.*  |
| Scala Play        | 2.5.*   |
| Scala Slick       | 3.2.0   |
| Java              | 8.*     |
| Docker            | 1.12    |
| PostgreSQL        | 9.6.*   |
